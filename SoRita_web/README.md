# SoRita_web

GitHub Pages için hazırlanmış statik site klasörü.

## İçerik

- `index.html`: ana sayfa
- `gizlilik.html`: gizlilik politikası
- `kvkk.html`: KVKK aydınlatma metni
- `cocuk-guvenligi.html`: çocuk güvenliği politikası
- `kullanim-kosullari.html`: kullanım koşulları
- `basvuru-kanallari.html`: doldurulması gereken resmi iletişim alanları

## Yayın öncesi kontrol

1. `basvuru-kanallari.html` içindeki köşeli parantezli alanları gerçek bilgilerle değiştir.
2. `kvkk.html` içindeki veri sorumlusu bilgilerini resmi unvan ve adresle güncelle.
3. Gerekirse tüm sayfalardaki `14 Nisan 2026` tarihini yeni güncelleme tarihiyle değiştir.
4. Klasör içeriğini doğrudan GitHub reposunun köküne yükle.

## GitHub Pages

Bu klasörde tüm linkler göreli olduğu için iki senaryo da çalışır:

- `username.github.io` tipi kullanıcı sayfası
- `repo-name` tipi proje sayfası

GitHub repo ayarlarında Pages kaynağını ana branch ve root klasörü olarak seçmen yeterlidir.
